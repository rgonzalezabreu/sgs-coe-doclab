Enum DecodingOption

Defines how metadata should be read back from Blob Storage. AutoDetect is a good choice if you trust the library. If a wrong type of decoding is specified the library does not check the suitability of the indicated encoding against the actual value in the blob's metadata.

Namespace: SGS.Framework.Azure.SimpleBlobStorage.Contracts

Name Description
AutoDetect

Default choice, leave the library to detect the encoding of metadata items and decode for you.
Base64

Decode the metadata originally stored in base64
DoNotDecode

Just returns the metadata in whatever format it is encoded and leaves the task to the caller.
PercentEncoding

Decode the metadata originally stored in uri style percent encoding
SafeUnicodeEscaping
